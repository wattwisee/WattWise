# WattWise

WattWise is an electricity usage dashboard that helps users monitor daily power consumption, estimate monthly electricity bills, and compare usage trends over time.

It uses the Gemini API to perform intelligent calculations, generate usage statistics, and suggest ways to reduce electricity bills. The Google Maps API detects the userâ€™s location to identify the correct electricity provider, ensuring accurate, personalized billing estimates for the Philippine market.

To stay current, WattWise gathers real-time pricing data from major providers such as Meralco through web scraping. Users receive notifications when electricity rates change, helping them make informed decisions about their energy consumption.
